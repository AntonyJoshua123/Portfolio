# Antony Joshua - Portfolio Website

A modern, professional one-page portfolio website for Full Stack Developer and IT Student Antony Joshua.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Smooth Scrolling**: Seamless navigation between sections
- **Modern UI**: Clean, professional design with tech-focused color palette
- **Interactive Elements**: Hover effects, animations, and smooth transitions
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Contact Integration**: Direct links to phone, email, LinkedIn, and GitHub

## Sections

1. **Hero/Home**: Introduction with profile image and call-to-action buttons
2. **About**: Professional summary and contact details
3. **Skills**: Technical and soft skills organized by category
4. **Projects**: Showcase of GitHub projects with descriptions
5. **Experience**: Internship experience timeline
6. **Contact**: Contact information and social links
7. **Footer**: Copyright and social media links

## Setup Instructions

1. Replace `Joshua_image.jpeg` with your actual profile image if needed
2. Update the resume file path in the download links if needed
3. Verify all contact information and links are correct
4. Open `index.html` in a web browser
